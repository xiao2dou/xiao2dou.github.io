使用阿里云服务器搭建VPN加密流量

Posted on 2020-08-18 Edited on 2020-08-23 In 技术

为了解决部分网站、邮件客户端未使用加密协议的问题，可以通过在云服务器上搭建VPN（Virtual Private Network，虚拟专用网），使用VPN来对敏感流量进行加密，从而达到保护个人信息的目的。

本文使用环境：Ubuntu 16.04

依赖：Python

开始安装：

  1. 安装shadowsocks

    $ pip install shadowsocks

  2. 配置shadowsocks

    创建shadowsocks.json配置文件

    $ vim /etc/shadowsocks.json

    配置文件内容：

    说明：

    • server：服务器 IP地址 （使用0.0.0.0表示本机)
    • server_port：服务器使用的端口，建议443，避免被查封（同时注意登陆阿里云控制台，在防火墙中添加443端口，包括tcp和udp两个协议类型）
    • password：建议修改为自己的密码

  3. 启动shadowsocks

    $ ssserver -c /etc/shadowsocks.json -d start

    关闭shadowsocks

    $ ssserver -d stop

  4. 检查服务器端口

    检查服务器相关端口是否在监听

    $ netstat -lnp

  5. 配置客户端
